FAQs

What is Frontend Mentor, and how will it help me?

Frontend Mentor is an online platform that provides you with real-world projects that you can use to practice your HTML, CSS, and JavaScript skills. Each project comes with a design to match and a brief to guide you through what you need to do. You can also compare your solution with others to see how they tackled the same project.
